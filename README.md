# Парсер книг с сайта tululu.org

Данная программа-парсер позволяет автоматизировать процесс скачивания книг из библиотеки [tululu.org](https://tululu.org/)

### Как установить

Python3 должен быть уже установлен. Затем используйте pip для установки зависимостей:

```pip install -r requirements.txt```

### Аргументы
Программа содержит сразу несколько исполняемых файлов - 
`tululu.py`  и `parse_tululu_category.py`.
Разница заключается в том, что скрипт `tululu.py`  скачивает все книги в заданном диапозоне (см. описание ниже),
а скрипт `parse_tululu_category.py` может скачивать целые страницы раздела "Научная фантастика" библиотеки.
Кроме того, `parse_tululu_category.py` является модифицированной версией и содержит большее число аргументов (см. описание ниже).

##### tululu.py
При запуске файла - `tululu.py` Вы можете использовать следующие аргументы:
* `--start_page` - ID (идентификатор) книги, с которой программа начнет скачивание. Сокращенная запись `-s`
* `--end_page` - ID (идентификатор) книги, на которой скачивание закончится (не включая её). Сокращенная запись `-e`

Пример для запуска скачивания книг с 1 по 5 ID:

```python tululu.py -s 1 -e 6```, что эквивалентно: 

```python tululu.py --start_page 1 --end_page 6```
##### parse_tululu_category.py

При запуске файла - `parse_tululu_category.py` Вы можете использовать следующие аргументы:
###### Все указанные аргументы являются необязательными.

* `--start_page` - Номер страницы, с которой программа начнет скачивание. Сокращенная запись `-sp`. Тип - int(целое число). По умолчанию 1.
* `--end_page` - Номер страницы, до которой программа проводит скачивание (не включая её). Сокращенная запись `-ep` Тип - int(целое число). По умолчанию 5
* `--destfolder` - путь до "рабочей" директории скрипта. Именно в данную директорию программа будет сохранять результат работы: книги и картинки к ним в соответствующие директории books, img, а также данные в формате json. Сокращенная запись `-df`. Тип - string(строка).
По умолчанию рабочей директорией является директория, где располагается скрипт `parse_tululu_category.py`
* `--no-downloadimg` - В случае запуска скрипта с данным аргументом (указывать дополнительные значения не нужно) программа не скачивает изображения обложек книг. По умолчанию скачивание включено.
* `--no-downloadtext` - В случае запуска скрипта с данным аргументом (указывать дополнительные значения не нужно) программа не скачивает текстовые файлы книг. По умолчанию скачивание включено.
* `--jsonpath` - путь до файла json, в котором будут записаны данные о результатах парсинга. При этом не нужно указывать расширение файла .json, скрипт добавит его автоматически. Сокращенная запись `-jp`. Тип - string(строка). 
По умолчанию файл сохранится в директорию со скриптом либо (в случае указания такой) в заданной в команде `--destfolder` директории. Имя файла по умолчанию - `books.json`.
